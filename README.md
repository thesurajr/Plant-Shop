# 🌿 Plant Shop - E-Commerce Webpage

A modern and responsive Plant Shop E-Commerce landing page built using **React.js** and **Tailwind CSS**. This project showcases a clean UI for an online plant store, allowing users to explore featured plants, categories, and promotional sections.

## 🚀 Live Demo

🔗 Live Website: https://plant-shop-92rp4axn8-suraj-mauryas-projects.vercel.app/

## 📖 About the Project

This is a **static e-commerce webpage** created for practice and portfolio purposes. The project focuses on responsive design, component-based development, and modern UI implementation using React and Tailwind CSS.

## 📂 Project Setup

Clone the repository:

```bash
git clone https://github.com/thesurajr/Plant-Shop.git
```

Navigate to the project folder:

```bash
cd Plant-Shop
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
